# Just Json
Just write/read json data to a file stream.

* 1 byte chars only. No unicode.
* Strings are escaped.
* Leans on FILE *
* No allocations (except for if the cstdlib decides to allocate)
* No formatting or indentation.

See files for usage and documentation.
